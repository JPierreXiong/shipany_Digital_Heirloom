- **问题**: 用户付费后无法获得会员权限

- **影响**: 💰 直接影响收入和用户满意度

### 2. 注册时自动创建 Vault 🟡 P1
- **问题**: 新用户需要手动访问设置页面才能创建保险箱
- **修复**: 在 Better-Auth 钩子中自动创建
- **影响**: 🎁 提升新用户体验

### 3. Free 用户试用期管理 🟡 P1
- **问题**: 无法区分试用中和试用过期
- **修复**: 注册时设置 7 天试用期（currentPeriodEnd = now + 7 days）
- **影响**: 📅 清晰的试用期管理

### 4. 受益人解密次数限制 🟡 P1
- **问题**: Free 用户可以无限次解密（违反设计）
- **修复**: 在解密 API 中严格检查次数限制
- **影响**: 🔒 保护商业模式

```

5. 验证可以添加 10 个受益人

```

#### 场景 3: Free 用户试用期
```

4. 7 天后显示"试用期已过期"
```
